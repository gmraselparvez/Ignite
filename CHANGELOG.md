# Changelog

All notable changes to Ignite. This project loosely follows [Semantic Versioning](https://semver.org/).

## [3.4] — 2026-07-06
- Overview stat cards now have a fixed height, so cards with two-line titles no longer change height when the window is resized.

## [3.3]
- Widened the menu-bar popover so task names have more room before truncating.

## [3.2]
- Increased font sizes app-wide for readability — small text gets the biggest bump; the large timer display is unchanged.

## [3.1]
- Schedule times are now typeable HH:MM fields instead of stepper-only.
- Settings controls share one consistent right-aligned column.

## [3.0]
- Scheduled blocking redesigned: day selector on its own row; compact time fields replace the long time dropdown.

## [2.x]
- Consistent native-style dropdowns across Tasks, Reports, and Settings.
- Version label now reflects the real app version.
- Reports & Invoices unified: project grouping, a shared "show task time" toggle, and A4 export.
- Report PDF now includes the bar and pie charts.
- Invoice layout reworked: Description / Hours / Rate / Amount, grouped by project with transparent billing.
- Distinct categorical chart colors.
- Pill-shaped buttons with hover and pressed states throughout.

## [1.x]
- Initial Ignite release: Pomodoro + manual timer, tasks/projects/clients, smart breaks,
  wellness reminders, distraction blocking, automatic meeting detection with AI minutes,
  reports, invoices, premium settings, local persistence, app and menu-bar icons, and
  launch-at-login.
